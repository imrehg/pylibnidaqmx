Method n.l.T.get_buffer_size(self, on_board=False):

Part of nidaqmx.libnidaqmx.Task View Source

Specifies the number of samples the I/O buffer can hold for each channel in the task. Zero indicates to allocate no buffer. Use a buffer size of 0 to perform a hardware-timed operation without using a buffer. Setting this property overrides the automatic I/O buffer allocation that NI-DAQmx performs.

If on_board is True then specifies in samples per channel the size of the onboard I/O buffer of the device.
