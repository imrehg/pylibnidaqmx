Method n.l.T.__del__(self, libnidaqmx=libnidaqmx):

Part of nidaqmx.libnidaqmx.Task View Source

Clears the task.

Before clearing, this function stops the task, if necessary, and releases any resources reserved by the task. You cannot use a task once you clear the task without recreating or reloading the task.

If you use the DAQmxCreateTask function or any of the NI-DAQmx Create Channel functions within a loop, use this function within the loop after you finish with the task to avoid allocating unnecessary memory.
