nidaqmx.DigitalInputTask.set_buffer¶

DigitalInputTask.set_buffer(samples_per_channel)¶

Overrides the automatic I/O buffer allocation that NI-DAQmx performs.

Parameters:

samples_per_channel : int

The number of samples the buffer can hold for each channel in the task. Zero indicates no buffer should be allocated. Use a buffer size of 0 to perform a hardware-timed operation without using a buffer.
Returns:

success_status : bool
