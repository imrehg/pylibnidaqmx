nidaqmx.libnidaqmx.make_pattern¶

nidaqmx.libnidaqmx.make_pattern(paths, _main=True)¶

Returns a pattern string from a list of path strings.

For example:

>>> make_pattern(['Dev1/ao1', 'Dev1/ao2','Dev1/ao3', 'Dev1/ao4'])

‘Dev1/ao1:4’
