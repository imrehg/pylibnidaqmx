Method n.l.T.configure_trigger_digital_edge_start(self, source, edge='rising'):

Part of nidaqmx.libnidaqmx.Task View Source

Configures the task to start acquiring or generating samples on a rising or falling edge of a digital signal.

Parameters:

source - The name of a terminal where there is a digital
signal to use as the source of the trigger.
edge - Specifies on which edge of a digital signal to start

acquiring or generating samples:

'rising' - Rising edge(s). 'falling' - Falling edge(s).
