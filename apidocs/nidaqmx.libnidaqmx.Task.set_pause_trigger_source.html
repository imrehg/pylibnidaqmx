Method n.l.T.set_pause_trigger_source(self, source):

Part of nidaqmx.libnidaqmx.Task View Source

Specifies the name of a virtual channel or terminal where there is an analog signal to use as the source of the trigger.

For E Series devices, if you use a channel name, the channel must be the only channel in the task. The only terminal you can use for E Series devices is PFI0.
