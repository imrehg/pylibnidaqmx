Method n.l.T.configure_trigger_digital_pattern_start(self, source, pattern, when='matches'):

Part of nidaqmx.libnidaqmx.Task View Source

Configures a task to start acquiring or generating samples when a digital pattern is matched.

Parameters:

source - Specifies the physical channels to use for pattern
matching. The order of the physical channels determines the order of the pattern. If a port is included, the order of the physical channels within the port is in ascending order.
pattern - Specifies the digital pattern that must be met for
the trigger to occur.
when - Specifies the conditions under which the trigger

occurs:

'matches' - Pattern matches. 'does_not_match' - Pattern does not match.
