Method n.l.T.set_buffer(self, samples_per_channel):

Part of nidaqmx.libnidaqmx.Task View Source

Overrides the automatic I/O buffer allocation that NI-DAQmx performs.

Parameters:

samples_per_channel - The number of samples the buffer can
hold for each channel in the task. Zero indicates no buffer should be allocated. Use a buffer size of 0 to perform a hardware-timed operation without using a buffer.
